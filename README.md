# Introduction
<div style="text-align:center;"><img src="https://raw.githubusercontent.com/wlhcode/hangman/gh-pages/img.png" alt="scrshot"></div>

This version of Hangman is a 2-player game, exclusively for Windows.

Hangman is a paper and pencil guessing game for two players. One player thinks of a word or phrase and the other tries to guess it by suggesting letters or numbers, within a certain number (9 in the case of this version) guesses. Learn more on [Hangman - Wikipedia](https://en.wikipedia.org/wiki/Hangman_(game)).
